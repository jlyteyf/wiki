chunk ​

语法

_.chunk(array, [size=1])

描述 ​

将数组拆分成多个 size 长度的块，并组成一个新数组。 如果数组无法被分割成全部等长的块，那么最后剩余的元素将组成一个块。

参数 ​

参数类型默认值说明
arrayArray-需要处理的数组
sizenumber1每个块的长度

返回值 ​

  • array (Array): 返回一个拆分好的新数组

示例 ​

js
_.chunk(['a', 'b', 'c', 'd'], 2);
// => [['a', 'b'], ['c', 'd']]

_.chunk(['a', 'b', 'c', 'd'], 3);
// => [['a', 'b', 'c'], ['d']]
